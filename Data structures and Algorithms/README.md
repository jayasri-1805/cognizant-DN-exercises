# Data Structures and Algorithms Lab

This repository contains the implementation of Data Structures and Algorithms laboratory exercises. Each exercise is based on a real-world scenario and demonstrates the application of fundamental data structures, algorithms, and complexity analysis using Java.

## Repository Objectives

- Understand fundamental data structures and algorithms.
- Implement real-world problem-solving applications.
- Analyze time and space complexity.
- Compare different algorithms based on performance.
- Improve programming and analytical skills.

## Exercises

### Exercise 1 – Inventory Management System
**Concepts Covered**
- Data Structures
- ArrayList / HashMap
- CRUD Operations
- Time Complexity Analysis

**Scenario**
Develop an inventory management system that efficiently stores and manages product details.

---

### Exercise 2 – E-commerce Platform Search Function
**Concepts Covered**
- Big O Notation
- Linear Search
- Binary Search
- Performance Comparison

**Scenario**
Implement and compare searching algorithms for an e-commerce platform.

---

### Exercise 3 – Sorting Customer Orders
**Concepts Covered**
- Bubble Sort
- Quick Sort
- Sorting Algorithms
- Complexity Analysis

**Scenario**
Sort customer orders based on total order value.

---

### Exercise 4 – Employee Management System
**Concepts Covered**
- Arrays
- Searching
- Traversal
- Deletion
- Array Representation

**Scenario**
Manage employee records using arrays.

---

### Exercise 5 – Task Management System
**Concepts Covered**
- Singly Linked List
- Dynamic Memory Allocation
- Linked List Operations

**Scenario**
Manage tasks efficiently using linked lists.

---

### Exercise 6 – Library Management System
**Concepts Covered**
- Linear Search
- Binary Search
- Searching Techniques

**Scenario**
Search books by title using different searching algorithms.

---

### Exercise 7 – Financial Forecasting
**Concepts Covered**
- Recursion
- Recursive Algorithms
- Optimization Techniques

**Scenario**
Predict future financial values using recursive methods.

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Data Structures
- Algorithms

## Learning Outcomes

After completing these exercises, you will be able to:

- Implement common data structures in Java.
- Apply searching and sorting algorithms.
- Analyze algorithm efficiency using Big O notation.
- Solve practical problems using recursion.
- Compare different approaches based on performance.
- Understand appropriate data structure selection for different scenarios.
