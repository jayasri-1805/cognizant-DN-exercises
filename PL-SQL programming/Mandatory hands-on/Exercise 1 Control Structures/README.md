# PL/SQL Exercise 1 - Control Structures

## Objective
This repository contains solutions for three PL/SQL control structure scenarios.

## Project Structure

```
PLSQL_Exercise1_GitHub/
│── sql/
│   ├── 01_schema.sql
│   ├── 02_sample_data.sql
│   ├── 03_scenario1.sql
│   ├── 04_scenario2.sql
│   └── 05_scenario3.sql
│── README.md
```

## Scenarios
1. Apply a 1% loan interest discount for customers older than 60 years.
2. Mark customers with balance greater than 10000 as VIP.
3. Display reminders for loans due within the next 30 days.

## Software Used
- Oracle Database
- Oracle SQL Developer / SQL*Plus

## Execution Order

1. Run `01_schema.sql`
2. Run `02_sample_data.sql`
3. Run `03_scenario1.sql`
4. Run `04_scenario2.sql`
5. Run `05_scenario3.sql`

Enable **DBMS Output** before executing the PL/SQL blocks.
